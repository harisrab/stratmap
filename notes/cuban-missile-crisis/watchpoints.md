## Analytic watchpoints

Use these prompts to move from a static map to an investigative workflow.

- Which geographic facts changed strategic perception more than military reality?
- How did the quarantine create a visible bargaining line without forcing an immediate general-war decision?
- Which moments in the timeline were driven by map facts and which by signaling and back-channel diplomacy?
- How did the same map support military readiness, intelligence interpretation, and public communication?
