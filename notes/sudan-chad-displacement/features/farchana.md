# Farchana camp system

Camp and services zone in eastern Chad representing the deeper humanitarian system beyond the immediate frontier.

This point shows the difference between arrival and stabilization. The corridor cannot be understood if all attention stays on the border strip and none on the inland camps carrying the longer burden.
