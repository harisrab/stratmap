# Cape of Good Hope detour

A simplified diversion line showing the long alternate path that some ships considered or took while the canal remained blocked.

This route is the map's reminder that every chokepoint has an outer substitute. The substitute is costly, but it is what gives operators leverage when the main line becomes unreliable.
