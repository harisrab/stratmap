## Analytic watchpoints

Use these prompts to move from a static map to an investigative workflow.

- Which relief routes remained functional in the first 72 hours and which only appeared open on a static map?
- How did cross-border politics shape humanitarian geography in northwest Syria?
- Where did shelter, health, and fuel demand concentrate faster than response capacity?
- How can the same map distinguish rescue, relief, and recovery phases instead of flattening them into one disaster layer?
